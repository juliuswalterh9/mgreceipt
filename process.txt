# 코드 개요
- 회사에서 법인카드로 비용을 쓰고, 영수증을 간소화 처리하는 프로세스 개발
- 현재는 카드 영수증을 제출하면, A4 용지에 풀로 붙이고, 스캔하여 회계 증빙에 사용

# 필요 모듈
- 로그인 모듈
- 사용자 프론트 모듈
- 관리자 백엔드 모듈

# 프론트 코드 프로세싱
- 음식점에서 카드 영수증을 받으면, 모바일로 서비스에 로그인하여 영수증 사진을 찍는다
- 승인 일시, 카드번호, 승인 금액, 사업자번호, 사용처(매장명), 사용자를 OCR 하고,
- 사용자에게 내역이 맞는지 확인, 수정하고, 확인을 누르면 데이터와 파일이 서버에 저장됨

# 백엔드 코드
- 관리자 화면에서 비용 사용내역을 조회
- 승인일시, 사용자로 조회 조건
- 클릭하면 상세 내역을 조회 (Open Window)
- 영수파일 다운로드 가능

# 로그인 모듈
- 사용자를 등록, 조회, 변경, 삭제 한다.
- Userid, Password, 사용자명, 사용자부서, 전화번호, 메일주소, 직책, 비고 등록 관리
